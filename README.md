# GrowteX Ventures Website

A premium, high-converting landing page for **GrowteX Ventures**, India's startup growth partner.

## 🚀 Features
- **Modern UI/UX**: Built with a sleek venture-capital aesthetic using Bebas Neue, Syne, and DM Sans.
- **Dynamic Animations**: Includes scroll-driven reveals, marquee text, and custom cursor interactions.
- **Premium Assets**: Custom-generated logo and ultra-high-definition hero background.
- **Interactive Chatbot**: "Arya", the AI agent, handles lead generation and service inquiries (requires backend integration for production).
- **SEO Optimized**: Complete meta tags and JSON-LD schema for maximum visibility.

## 📁 Project Structure
```text
growtex-ventures/
├── index.html          # Core structure and SEO
├── css/
│   └── style.css       # Design system and animations
├── js/
│   └── script.js       # Interactive logic and Chatbot
└── assets/
    ├── logo.png        # Premium company logo
    └── hero-bg.jpg     # High-end background image
```

## 🛠️ Tech Stack
- **HTML5 / CSS3**: Vanilla implementation for maximum performance and control.
- **JavaScript (ES6+)**: Custom logic for interactions and API calls.
- **Anthropic Claude API**: Powering the chatbot (Arya).

## 📝 Developer Notes
- **API Security**: The chatbot currently makes a client-side fetch. For production use, please move the Anthropic API call to a secure backend proxy to protect your API keys and avoid CORS issues.
- **Customization**: Update the WhatsApp links (`https://wa.me/917999866007`) to your business number.
